
# 🎓 Automated Student Attendance Monitoring and Analytics System for Colleges

## 📌 Project Overview

The Automated Student Attendance Monitoring and Analytics System is a smart web-based application designed to automate attendance management in colleges. It replaces traditional paper-based attendance with a digital system that records, manages, and analyzes student attendance efficiently.

The system helps faculty mark attendance quickly, enables students to view their attendance records, and provides administrators with detailed reports and analytics.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/attendance-system.git
```

### 2. Open Project

```bash
cd attendance-system
```

### 3. Configure Database

- Create a MySQL database.
- Import `attendance.sql`.
- Update database credentials.

### 4. Run the Project

For Flask:

```bash
python app.py
```

For Django:

```bash
python manage.py runserver
```
